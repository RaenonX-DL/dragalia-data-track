# 失落的龍絆 遊戲相關資料 & 追蹤

若有修改任何本Repo內的檔案，請附上來源。


## `Dragalia Data.xlsx`

這份EXCEL檔案追蹤手邊資源和對應資源掉落。詳情請下載檔案查看內容。

檔案內的 `導引` 工作表提供部分工作表的使用方式。


## `CEX.xlsx`

所有角色的CEX(連鎖EX)。


## `SS.xlsx`

所有角色的共享技能及相關資訊。


### Excel下載方法

#### 電腦
點選對應Excel檔案後，點選右上角`Download`下載。

#### 手機
點選對應Excel檔案後，點選內容區塊的`Open binary file`下載。
